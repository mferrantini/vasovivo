# Stampa 3D — VasoVivo

Questa cartella raccoglie i file per la stampa 3D dei componenti meccanici del progetto **VasoVivo**.
Ogni sottocartella corrisponde a un modello di contenitore o a una variante generica e contiene:

- `README.md` — descrizione, compatibilità, note di stampa e montaggio
- File STL (da aggiungere) — modelli pronti per lo slicer

---

## Indice modelli

| Cartella | Modello | Compatibilità |
|---|---|---|
| [`generic-lid/`](generic-lid/) | Coperchio generico | Barattoli con diametro regolabile o adattabile |
| [`eklatant/`](eklatant/) | Coperchio per EKLATANT | [IKEA EKLATANT](https://www.ikea.com/it/it/p/eklatant-contenitore-con-coperchio-vetro-trasparente-bambu-50621766/) |

---

## Note generali

- I coperchi ospitano la sensoristica e la scheda ESP32: verifica le tolleranze in base al tuo filamento e alla calibrazione della stampante.
- Per il barattolo di riferimento del progetto si consiglia il modello **EKLATANT**.
- I file STL verranno pubblicati nelle rispettive cartelle non appena disponibili.
