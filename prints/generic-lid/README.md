# Coperchio generico

Coperchio intelligente **generico**, progettato per adattarsi a barattoli di dimensioni simili tramite parametri o adattamenti manuali.

Utile per sperimentare con contenitori diversi dal modello di riferimento IKEA EKLATANT.

---

## Compatibilità

- Barattoli in vetro con bocca circolare di diametro compatibile (da verificare sul modello STL)
- Non ottimizzato per un singolo prodotto del catalogo
