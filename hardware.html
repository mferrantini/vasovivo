VasoVivo

Componenti hardware del progetto

Elenco della scheda e dei sensori utilizzabili in VasoVivo. Per ogni sensore trovi nella repository documentazione, schema di collegamento e uno sketch di esempio per ESP32.

  • 🔌 ESP32-WROOM-32

    Microcontrollore · Wi-Fi · Bluetooth

    Scheda principale del progetto: collega i sensori e gestisce lettura e invio dei dati.

    Acquista su Amazon →

  • 🌿 ENS160 + AHT21

    CO₂eq · TVOC · temperatura · umidità aria

    Modulo combinato per monitorare qualità dell'aria e condizioni ambientali nel barattolo.

    Documentazione su GitHub → Acquista su Amazon →

  • 🌡️ DHT22

    Temperatura · umidità aria

    Alternativa consigliata per misurare temperatura e umidità dell'aria con maggiore precisione.

    Documentazione su GitHub → Acquista su Amazon →

  • 🌡️ DHT11

    Temperatura · umidità aria

    Alternativa economica per temperatura e umidità dell'aria, adatta a un primo approccio al progetto.

    Documentazione su GitHub → Acquista su Amazon →

  • 💧 Sensore a forchetta

    Umidità substrato

    Misura direttamente l'umidità della zolletta per capire se il substrato è troppo secco o troppo bagnato.

    Documentazione su GitHub → Acquista su Amazon →

  • 🌱 DS18B20

    Temperatura substrato

    Sonda impermeabile per la temperatura interna al substrato, complementare alla temperatura dell'aria.

    Documentazione su GitHub → Acquista su Amazon →

  • ☀️ Fotoresistenza (LDR)

    Luminosità

    Rileva la luce ambientale per verificare l'esposizione e correlarla con l'andamento della coltivazione.

    Documentazione su GitHub → Acquista su Amazon →
